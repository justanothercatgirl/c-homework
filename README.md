# Файлы
* main.c: главный файл
* Makefile: система сброки GNU Make
* libs/ : папка с библиотеками для теста. а каждом файле там по 1 функции.

# Как протестировать
## Требования
* ОС: Unix
* gnu make 
* командная строка
## Сборка
* cd <путь к этому файлу>
* make all
## Запуск
`./main ./libsq `. вместо libsq может быть libsin, libpoly
